#Автотесты для проверки сайта Яндекс.Самокат

##Все тесты хранятся в директории tests:

test_main_page.py - Содержит тесты проверяющие элементы главной страницы

test_order_page.py - Содержит тесты проверяющие элементы страницы заказа

test_url_page.py - Содержит тесты проверяющие логотипы на главной странице

##В папке pages хранятся файлы с функциями:

base_page.py - Содержит базовые функции для работы со всеми элементами

main_page.py - Содержит функции для элементов главной страницы

order_page.py - Содержит функции для элементов страницы заказа

##В папке Locators хранятся файлы с локаторами страниц:

base_locators.py - Содержит локаторы логотипов

main_page_locators.py - Содержит локаторы с главной страницы

order_page_locators.py - Содержит локаторы со страницы заказа
